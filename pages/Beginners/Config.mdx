# Configs and Profiles in Raven Bs

In Raven Bs, configs are referred to as profiles.

### What Are Profiles?
Profiles let you save your settings so you can easily load them later. This feature is handy when you have a preferred setup for certain activities.

### Creating a Profile
<Steps>
### Step 1
Navigate to your <a href="/Modules/Client/GUI">GUI</a> and select the profile settings.
<Image src="/beginner_1.png" width={212} height={116} />
<Callout type="info">The default shortcut for accessing the GUI is `RSHIFT`.</Callout>
### Step 2
Right-click on "Manager" and select "Create Profile".
</Steps>

### Loading a Profile
<Steps>
### Step 1
Go to your <a href="/Modules/Client/GUI">GUI</a> and select the profile settings.
<Image src="/beginner_1.png" width={212} height={116} />
<Callout type="info">Remember, the default shortcut for the GUI is `RSHIFT`.</Callout>
### Step 2
Right-click on the profile you wish to load.
</Steps>

### Profile Storage Location
Your profiles are stored in the `keystrokesmod` folder, located in your `.minecraft` directory.
<div>
  <Callout type="info">
    To access your `.minecraft` directory, type `%appdata%.minecraft` here

  </Callout>
</div>
